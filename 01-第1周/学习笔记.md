# 4.13

## 基础语法、数据类型、控制流、函数、作用域

### 导入模块
- `import`
- `from ... import`

### 多行注释与续行
- `''''''` / `""""""` **这种注释方式不能嵌套使用**。
- 使用反斜杠 `\` 来实现多行语句。
- 在 `[]`、`{}`、`()` 中的多行语句，不需要使用反斜杠 `\`。
- 使用 `r` 可以让 `\` 不发生转义。

### 字符串与输出
- 按字面意义级联字符串，如 **"this " "is " "string"** 会被自动转换为 **this is string**。
- 字符串可以用 `+` 运算符连接在一起，用 `*` 运算符重复。
- `print` 默认输出是换行的，如果要实现不换行需要在变量末尾加上 `end=""`。

### 变量与类型
- Python 中的变量不需要声明。
- 允许同时为多个变量赋值：
  - `a = b = c = 1`
  - `a, b, c = 1, 2, "runoob"`
- 通过 `type()` 函数查看变量的类型。

### 6 种标准数据类型
- **Number**
  - `int`, `float`, `bool`, `complex`
  - `complex: a+bj / complex(a,b)`
  - 数值的除法包含两个运算符：`/` 返回一个浮点数，`//` 返回一个整数（向下取整）。
- **String**
  - 字符串截取：`变量[头下标:尾下标]`，不含尾下标。
  - Python 字符串不能被改变。
- **bool**
  - 以下值转换为布尔值时为 `False`：`None`、`False`、零（`0`、`0.0`、`0j`）、空序列（如 `''`、`()`、`[]`）和空映射（如 `{}`）。其他所有值转换为布尔值时均为 `True`。
  - `and`、`or`、`not`
- **Tuple**
  - 可以把字符串看作一种特殊的元组。
- **List**
  - 截取可以接收第三个参数，参数作用是截取的步长。
- **Set**
  - 集合中的元素不会重复，并且可以进行交集 `&`、并集 `|`、差集 `-`、对称差集 `^` 等常见的集合操作。
  - 创建一个空集合必须用 `set()` 而不是 `{}`，因为 `{}` 创建的是一个空字典。
- **Dictionary**
  - **键(key) : 值(value)**
  - 键(key) 必须使用不可变类型，且在同一个字典中键必须是唯一的。
  - Python 3.7 起，字典会保持元素的**插入顺序**，不再是无序的。
  - 构造函数 `dict()` 可以直接从键值对序列中构建字典。
- **bytes**
  - 不可变的二进制序列。
  - 元素是整数值（0 到 255 之间的整数），而不是 Unicode 字符。
  - 创建 bytes 对象最常见的方式是使用 `b` 前缀。
  - 也可以使用 `bytes()` 函数将其他类型的对象转换为 bytes 类型，第二个参数指定编码方式。
  - bytes 类型中的元素是整数值，因此在进行比较操作时需要使用相应的整数值。

- 用 `isinstance(Obj, Type)` 来判断数据类型。

> [!NOTE]
> `isinstance` 和 `type` 的区别在于：
> - `type()` 不会认为子类是一种父类类型。
> - `isinstance()` 会认为子类是一种父类类型。

- 可以通过 `is` 来判断对象身份。

### 数据类型转换
- `repr(x)` 将对象转化为供解释器读取的形式。
- `eval(str)` 将字符串 expression 解析为 Python 表达式，并在指定的命名空间中执行它。
  - `eval(expression[, globals[, locals]])`
- `frozenset(s)` 返回一个冻结的集合，冻结后集合不能再添加或删除任何元素。
- `chr()` 返回一个对应的字符。
- `ord()` 将一个字符转换为它的整数值（Unicode 码点）。
- `hex()` 将一个整数转换为十六进制字符串。
- `oct()` 将一个整数转换为八进制字符串。

### 运算符
- 海象运算符（`:=`）允许在表达式内部进行赋值，这可以减少代码的重复，提高代码的可读性和简洁性。
- `&` 按位与运算符
- `|` 按位或运算符
- `^` 按位异或运算符
- `~` 按位取反运算符
- `<<` 左移动运算符
- `>>` 右移动运算符
- `in`
- `not in`
- `is`
- `is not`

### 其他
- 数学函数
- 随机数函数
- 三角函数

## 列表/元组/字典/集合及其方法、推导式、切片

### 字符串切片
- `str[start:end:step]`

### 推导式
- `[expression for item in Sequence if conditional]`
- `{key_expr: value_expr for value in collection if condition}`
- `{expression for item in Sequence if conditional}`
- `(expression for item in Sequence if conditional)`

### 列表
- `len(list)`
- `max(list)`
- `min(list)`
- `list.append(obj)` 向队列的末尾添加一个元素。
- `list.pop([index=-1])` 移除列表中的一个元素（默认最后一个元素），并且返回该元素的值。
- `list.count(obj)` 统计某个元素在列表中出现的次数。
- `list.extend(seq)` 用新列表扩展原来的列表。
- `list.index(obj)` 从列表中找出某个值第一个匹配项的索引位置。
- `list.insert(index,obj)` 将对象插入列表。
- `list.remove(obj)` 移除列表中某个值的第一个匹配项。
- `list.reverse` 反向列表中元素。
- `list.sort(key=None,reverse=False)` 对原列表进行排序。
- `list.clear()` 清空列表。
- `list.copy()` 复制列表。
- `operator.eq(a,b)` 列表是否相等。

### 元组
- 值不允许被修改，但是可以对元组进行拼接组合。
- `del tuple` 删除整个元组。

### 字典
- 键必须是唯一的，但值则不必。
- 使用内建函数 `dict()` 创建字典。
- `len(dict)`
- `dict.clear()` 清空字典。
- `str(dict)` 输出字典，可以打印的字符串表示。
- `dict.fromkeys(seq[, value])` 返回一个新字典。
- `dict.items()` 以列表返回视图对象。
  - 配合 `for` 循环使用，在一次循环中，同时拿到键（key）和值（value）。
- `dict.keys()`
- `dict.values()`
- `dict.get(key[, value])` 返回指定键的值，如果键不在字典中返回默认值，如果不指定默认值，则返回 **None**。
- `dict.setdefault(key, default=None)` 与 `get()` 类似，但是如果键不存在于字典中，将会添加键并将值设为默认值。
- `dict.pop(key, default)`
- `dict.popitem()` 返回并删除字典中的最后一对键和值。

### 集合
- `set.add()` 添加元素。
- `set.update()` 添加元素，且参数可以是列表、元组、字典等；x 可以有多个，用逗号分开。
- `set.remove(x)` 删除元素，如果元素不存在发生错误。
- `set.discard(x)` 删除元素，如果元素不存在不发生错误。
- `set.pop()` 随机删除一个元素。
- `set.difference(set)` 返回多个集合的差集。
- `set.difference_update(set)` 用于移除两个集合中都存在的元素，没有返回值。
- `set.intersection(set1,set2 ... etc)` 返回集合的交集。
- `set.intersection_update(set1, set2 ... etc)` 在原始的集合上移除不重叠的元素。
- `set.isdisjoint(set)` 用于判断两个集合是否包含相同的元素，如果没有返回 True，否则返回 False。
- `set.issuperset(set)` 判断指定集合的所有元素是否都包含在原始的集合中，如果是则返回 True，否则返回 False。
- `set.remove(item)` 移除指定元素。
- `set.pop()` 随机删除元素。
- `set.symmetric_difference(set)` 返回两个集合中不重复的元素集合。
- `set.union(set1, set2...)` 返回两个集合的并集。
- `set.update(set)` 添加新的元素或集合到当前集合中。

## 字符串处理、格式化（f-string）、常用内置函数（map/filter/zip/enumerate/sorted）

### 字符串格式化
- `%s` 字符串
- `%d` 整数
- `%u` 无符号整型
- `%f` 浮点数字
- `%e` 科学计数法
- `%E` 科学计数法
- `%g` `%f` 和 `%e`
- `%G` `%f` 和 `%E`

### f-string（格式化字符串字面量）
- `f"{}"`
- `"{0}{1}".format(a,b)` 在括号中的数字用于指向传入对象在 `format()` 中的位置。

### 常用内置函数
- `str.capitalize()` 将字符串的第一个字母变成大写，其他字母变小写。
- `str.count(sub, start=0, end=len(string))` 返回 str 在 string 里面出现的次数。
- `str.find(str, beg=0, end=len(string))` 检测 str 是否包含在字符串中。
- `str.index(str, beg=0, end=len(string))` 跟 `find()` 方法一样，只不过如果 str 不在字符串中会报一个异常。
- `str.isalnum()` 检查字符串是否由字母和数字组成。
- `str.isalpha()` 如果字符串至少有一个字符并且所有字符都是字母或中文字则返回 True，否则返回 False。
- `str.isdigit()` 如果字符串只包含数字则返回 True，否则返回 False。
- `str.islower()` 如果字符串中包含至少一个区分大小写的字符，并且所有这些字符都是小写，则返回 True，否则返回 False。
- `str.isupper()`
- `str.isnumeric()`
- `str.istitle()` 检测字符串中所有的单词拼写首字母是否为大写，且其他字母为小写。
- `str.isspace()`
- `str.join(sequence)` 将序列中的元素以指定的字符连接生成一个新的字符串。
- `str.lower()` 转换字符串中所有大写字符为小写。
- `str.upper()`
- `str.strip([chars])` 移除字符串头尾指定的字符（默认为空格）或字符序列。
- `str.replace(old, new[, max])` 把字符串中的 old（旧字符串）替换成 new（新字符串），替换不超过 max 次。

### enumerate
- 获得元素和索引。
- `for index, item in enumerate(iterable):`
- 可以使用 `start` 更改索引的起始数字。
- `for num, item in enumerate(iterable, start=1):`
- 把列表转换成字典，键是索引（序号），值是元素。
  - `my_dict = dict(enumerate(iterable, start=0))`

### zip
- 把两个或多个列表里**对应位置**的元素打包在一起。
- **木桶效应**：如果列表长度不一，会以最短的那个为准，多余的会被丢弃。
- `for item1, item2 in zip(list1, list2):`
- 快速生成字典：
  - `my_dict = dict(zip(keys_list, values_list))`
- 解包：
  - `list1, list2 = zip(*zipped_data)`
- 保留最长的列表，缺少的用 `None`（或其他默认值）填补：

```python
from itertools import zip_longest
for item1, item2 in zip_longest(list1, list2, fillvalue="默认值"):
    ...
```

### sorted
- `sorted` 不会修改原数据，而是生成一个全新的列表，并且它能对所有可迭代对象起作用。
- `new_list = sorted(iterable)`
- `new_list = sorted(iterable, reverse=True)`
- `new_list = sorted(iterable, key=自定义规则函数)`
- 对字典排序：
  - `sorted_by_key = dict(sorted(my_dict.items(), key=lambda x: x[0]))`

### map()
- 在 Python 3 中，`map()` 得到的是一个“加工图纸”（map 对象），必须在外面包一层 `list()` 才能把结果真正生产出来展示给你。
- `new_list = list(map(转换类型, iterable))`
- `new_list = list(map(lambda x: 运算逻辑, iterable))`
- `new_list = list(map(函数名, iterable))`

### filter()
- 和 `map()` 一样，在 Python 3 中 `filter()` 得到的是一个对象，必须在外面包一层 `list()` 才能看到最终过滤好的列表。
- `new_list = list(filter(lambda x: 筛选条件(返回True/False), iterable))`
- 清除所有“假值”：
  - `clean_list = list(filter(None, iterable))`

### 迭代器
- `iter()`
- `next()`

### 生成器
- `yield`
- 调用一个生成器函数，返回的是一个迭代器对象。

## 文件读写、异常处理、模块与包

### 文件读写
- `with open() as f`
- 处理中文字符时，强烈建议加上 `encoding='utf-8'`；处理二进制数据 `rb/wb` 时不需要加 `encoding`。
- `open(filename, mode)`

- `f.read()`
- `f.readline()`
- `f.readlines()`
- `f.write()`
- `f.tell()` 返回文件当前的读/写位置（即文件指针的位置）。
- `f.seek(offset, from_what)` 改变文件指针当前的位置。
- `f.close()`

### pickle 模块
- 文件后缀名会习惯性写成 `.pkl`。
- **带 `s` 的处理变量，不带 `s` 的处理文件。**
- 序列化：
  - `pickle.dumps(obj)`：把对象转成 bytes（二进制流）。
  - `pickle.dump(obj, file)`：把对象转成 bytes 并**直接写入文件**。
- 反序列化（解冻）：
  - `pickle.loads(bytes_obj)`：把 bytes 还原成对象。
  - `pickle.load(file)`：**直接从文件读取** bytes 并还原成对象。

### 其他模块
- `json` 模块--类似pickle模块
- `os` 模块

### 异常处理
- 语法错误：`SyntaxError`
- `try/except` 语句：
  - `try:`
  - `except Exception as e:`
  - `else`
  - `finally`
- 一个 `except` 子句可以同时处理多个异常，这些异常将被放在一个括号里成为一个元组。
- 抛出异常：
  - `raise + 异常类型 + ("具体的错误提示语")`
- 有时候你在 `except` 里面捕获了异常，只是为了打印一下日志，但你**不想吞掉这个错误**，而是想让上层代码继续处理它，这时候可以单独写一个 `raise`。
