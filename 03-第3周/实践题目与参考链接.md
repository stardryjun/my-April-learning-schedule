## 二、小项目（周内完成并产出可运行示例）

### 项目：线性数据结构库（LinearDS）
- 要求：实现一套轻量的线性数据结构库，包含：动态数组（简化 vector）、单链表、双链表、栈、队列、循环队列、双端队列、单调栈辅助函数。
- 语言：提供 Python 版本和 C++ 版本（可互相对照）。
- 功能：每个结构包含常用方法、边界检查、时间复杂度注释、单元测试与示例脚本。
- 产出：
  - examples/ 目录下提供使用示例（run_examples.py / run_examples.cpp）
  - tests/ 目录提供基础单元测试（pytest for Python；简单的 GoogleTest-like 要求或一组手写测试脚本用于 C++）
  - README.md：使用说明与复杂度一览表
- 验收标准：所有核心方法通过示例脚本，且有 README 中复杂度说明与样例性能对比（示例数据量 1e5 / 1e6 视实现而定）。
