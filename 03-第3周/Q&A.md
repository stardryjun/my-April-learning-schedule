# Q1：链表使用时什么时候使用Node，什么时候使用指针
| 使用场景              | 使用 `Node`    | 使用 `Node*`（指针） |
| ----------------- | ------------ | -------------- |
| 定义链表的基本节点         | ✔            |                |
| 创建新节点到链表中         |              | ✔              |
| 修改链表的链接关系         |              | ✔              |
| 遍历链表              |              | ✔              |
| 插入、删除节点           |              | ✔              |
| 临时性独立的节点数据（非动态分配） | ✔            |                |
| 参数传递              |              | ✔              |
| 静态链表（构造在栈上的链表）    | ✔（例如数组的模拟链表） |                |
## 使用Node：
**节点的定义与初始化**
**临时存储数据**
**栈分配内存**
## 使用指针：
**链表的实际操作**（几乎所有链表操作都用指针）
**动态内存分配**
**修改链表链接关系**
**传递链表参数**
**操作大型链表数据结构**
